# 📄 Resume Analyzer

A simple and interactive Resume Analyzer built using HTML, CSS, and JavaScript. This project analyzes a user's resume against a job description, highlights missing keywords, and provides a match score along with suggestions to improve the resume.

## 🚀 Features

- Upload resume
- Paste job description
- Resume match score
- Keyword analysis
- Missing skills detection
- Resume improvement suggestions
- Clean and responsive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
Resume-Analyzer/
│── resume.html
│── resume.css
│── resume.js
```

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Shreyanshtanwar/resume-analyzer.git
```

2. Open the project folder.

3. Open `resume.html` in your browser.

No installation or dependencies are required.
